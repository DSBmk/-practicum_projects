# -practicum_projects
Этот репозиторий содержит учебные проекты, выполненные в рамках программы подготовки аналитика данных (Data Analyst). Проекты включают исследовательский анализ данных, статистический анализ, проверку гипотез, A/B-тестирование и визуализацию данных с использованием Python и его библиотек.
